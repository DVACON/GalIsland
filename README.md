### 欢迎使用GirlAndLove——galgame管理工具
✨ 功能特点

🎮 游戏库管理 - 添加、编辑、删除游戏，支持多种排序和筛选方式
🔍 VNDB 集成 - 自动从 VNDB 获取游戏信息、封面、截图、评分等
🏷️ 智能标签 - 从 ggbases 自动获取中文标签
🌐 AI 翻译 - 使用 DeepSeek AI 自动翻译游戏简介
🖼️ 图片管理 - 本地存储游戏封面和截图，支持拖拽上传
📁 分类管理 - 自定义分类，灵活组织游戏库
